Every sub-agent dispatched via the `Agent` tool gets a human-readable codename prefixed in the description. Two tiers, both rotating.

**Tier 1; first-class roster** (roles listed in `ai/swarm/README.md`; 12 slots like issue-writer, pr-describer, refactor-planner, save-format-warden, etc.):
- Roles are permanent. Codenames rotate per work unit (each issue/research-thread/challenge/design drop).
- When a work unit opens, pick a fresh name for each agent involved, chosen to fit the **specific case's personality**. A whimsical tutorial-copy challenge might see Slartibartfast as pr-describer; a tense save-migration challenge might see Marvin instead. Same role (pr-describer), different name, chosen for flavour.
- When the work unit closes and `ai/swarm/` scrubs (see `feedback_scrub_agents_on_completion.md`), all names release. Next work unit picks anew.
- Source pool: Gravity Falls, Hitchhiker's Guide, Oddworld (full cast: heroes like Abe, Munch, the Stranger, Toby, the Shaman; Mudokons generally; villains the Glukkons; Molluck, Lulu, Aslik, Dripik, Phleg, Morguer; plus Vykkers; the three named individual Sligs per fiction are Skillya, Slig Mama, and the Chauffeur; avoid "Slig" as a codename since it's the species, not an individual), Omori, Outer Wilds (Hearthians like Gabbro, Feldspar, Riebeck, Hornfels, Chert, Hal, Solanum; Nomai like Escall, Poke, Pye, Annona), **Simpsons** (Bart, Lisa, Marge, Homer, Maggie, Moe, Apu, Krusty, Milhouse, Ralph, Skinner, Flanders, Burns, Smithers, Otto, Edna, Nelson, Barney, Lenny, Carl, Wiggum, Quimby, Frink, Hibbert, Selma, Patty, Cletus, Disco Stu; vast cast, pick personality-fit), **Minions** (named individuals: Kevin, Bob, Stuart, Jerry, Mel, Tom, Dave, Carl, Tim, Phil; avoid "Minion" as a codename since it's the group label).

**Tier 2; on-demand / ad-hoc dispatches** (research spikes, one-off probes):
- Same personality-match rule: pick for fit, not at random.
- Preferred source: Volley's own cast. As of 2026-04-21 only **Martha** is named. Fallback: same five fictional universes.
- Topic-sticky within a session: follow-ups on the same thread reuse the original name even when spawning a fresh agent.

**Why:** Named handles let Josh track agents by vibe, not by process. Rotating per work unit keeps the names feeling fresh and tied to the case, rather than a stale fixture. Personality-matching lets the name add signal; Marvin on a root-cause for a nasty regression tells Josh the agent is coming in prepared for bad news; Kel on a burst of new test-writing tells him the vibe is energetic and prolific. Flagged 2026-04-20; rotation + personality-match rule agreed 2026-04-21.

**How to apply:**
- Put the name at the front of the agent `description`: `"Dipper: draft SH-42"`, `"Marvin: why does speed-bar drift on reload"`.
- **When dispatching a fresh agent, tag the source universe in the user-facing sentence for fun.** Example: "Dispatching **Hornfels** (Outer Wilds chief astronomer) on ...". Keeps the pool legible to Josh without him having to recall which show every name came from. Short, one parenthetical, never in the agent's own prompt.
- Pick names that fit the *case*, not just the role. When dispatching an issue-writer, read the subject matter first; melancholy feature, comic one, security-heavy, exploratory; and pick accordingly.
- Across one work unit, a given role gets one name. Don't reassign mid-unit; that destroys the handle.
- **Names are unique within a swarm.** No name appears twice in the current active cast; not across roles, not across Tier 1 and Tier 2 dispatches. Each handle resolves to exactly one agent while the swarm is live. Reuse only becomes legal after the swarm scrubs.
- Avoid themes reserved for other things: Linear cycles use alphabetical puppets. Do not cross the streams.
- No real public figures, colleagues, or anyone Josh might know.
- **Uniqueness check is a dispatch-time trigger, not a rule-to-remember.** Before every new `Agent` call, scan in-flight dispatches for codename collision. If the preferred name is live, pick a different one. Applies especially to parallel same-role dispatches (two docs-and-writing agents in flight → two different minion names, never two Daves). Reinforced 2026-04-23 after both parallel docs agents on #339 and #340 got tagged `[dave]`, collapsing the handle for Josh's review trail.
- **Named individuals only, never species or group labels.** "Slig" is the species; use a named Slig (Flunk, commander names, etc.). "Minion" is the group; use Kevin, Bob, Stuart, Jerry, Mel, Tom, Dave, Carl, Tim, Phil. "Hearthian" is the species; use Gabbro, Feldspar, Riebeck, Hornfels, Chert, Hal, Solanum. "Nomai" is the civilisation; use Escall, Poke, Pye, Annona, etc. If I don't know a specific named individual for the case, pick from a pool I do know rather than default to the group label.
