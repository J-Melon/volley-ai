In Volley's item design, only powerful items get names. Ordinary items, the starter set among them, stay unnamed and are referred to by a descriptor heading ("Ball: split-and-merge", "Helmet", "Magnetism", "Glove: pick").

**Why:** 2026-05-27. I proposed names for all six starter items; Josh: "never mind only powerful items get names." A name signals significance, so reserving names for powerful items keeps that signal meaningful; an unnamed item reads as ordinary by its very lack of a name.

**How to apply:** use descriptor headings for ordinary starter items; reserve naming for powerful items. Pairs with [[feedback_describe_before_naming]] and [[feedback_starter_items_no_bark_tables]].
