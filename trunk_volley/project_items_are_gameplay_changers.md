Every Volley item effect is a unique change to how the game plays: a new behaviour, a rule shift, a causality the player can read through play. Not "use it and a number goes up." `designs/01-prototype/design/items.md` says effects are gameplay-first and causality-driven, and passive stat boosts are "a subset"; do not build a set out of that subset. The existing items are the model (The Stray spawns balls and triggers frenzy, The Call flips colour multipliers and deflects, Dead Weight warps trajectory), not the simple stat items (Ankle Weights, Grip Tape).

This holds for STARTER items too. "Simple" means easy to read and one clean effect, not "a stat with a multiplier." A starter item is a simple unique gameplay-change, not a tame number.

An item effect is not a reskin of a core mechanic. Grabbing/catching the ball, hitting with the racquet, levitating, positioning are core, available to everyone; an item does not "let you" do those. It adds a NEW reactive effect (a trigger fires an outcome, like the causality items) OR a genuinely new ability that is not already core. The test is whether the thing is already core, not whether it is an action. (2026-05-25: a glove that "lets you catch and throw" was rejected, grabbing is core; a helmet that "lets you header the ball" was accepted, heading is a new contact, not core.)

**Why:** 2026-05-25, designing the starter set. I kept proposing usage-to-number effects (reach consolidation faster, more soul per hit, more soul by height); Josh: "i feel usage makes number go up but item effects are unique, something that changes gameplay." The starter set must be unique gameplay-changers, drawn simple. Sharpens the gameplay-first line already in items.md.
