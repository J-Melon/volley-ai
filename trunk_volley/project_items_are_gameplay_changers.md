Every Volley item effect is a unique change to how the game plays: a new behaviour, a rule shift, a causality the player can read through play. `designs/01-prototype/design/items.md` says effects are gameplay-first and causality-driven; passive stat boosts occupy "a subset" of the design space. Build from the model items (The Stray spawns balls and triggers frenzy, The Call flips colour multipliers and deflects, Dead Weight warps trajectory); the simple stat items (Ankle Weights, Grip Tape) mark the subset's edge.

This holds for STARTER items too. "Simple" means easy to read with one clean effect, distinct from a stat with a multiplier. A starter item is a simple unique gameplay-change, built around a single clean effect rather than a tame number.

An item effect adds something beyond a core mechanic. Grabbing and catching the ball, hitting with the racquet, levitating, and positioning are core mechanics available to everyone; an item adds a NEW reactive effect (a trigger fires an outcome, like the causality items) OR a genuinely new ability beyond the core set. The test asks whether the thing is already a core mechanic; an action earns its place when it adds a new ability beyond the core set. (2026-05-25: a glove that "lets you catch and throw" was rejected because grabbing is core; a helmet that "lets you header the ball" was accepted because heading is a new form of contact beyond the core set.)

**Why:** 2026-05-25, designing the starter set. I kept proposing usage-to-number effects (reach consolidation faster, more soul per hit, more soul by height); Josh: "i feel usage makes number go up but item effects are unique, something that changes gameplay." The starter set must be unique gameplay-changers, drawn simple. Sharpens the gameplay-first line already in items.md.
