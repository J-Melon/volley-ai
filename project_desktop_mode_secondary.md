Desktop companion mode (small borderless window on desktop) has been pushed to a secondary mechanic. Fullscreen is now the main way to play Volley!. The desktop mode still exists but should not be positioned as the primary hook in marketing or design discussions.

**Why:** Design decision made during prototype phase (April 2026). The fullscreen experience is the main draw.

**How to apply:** Don't frame Volley! as a "desktop companion game" or compare it primarily to Rusty's Retirement. Position it as a cozy idle game with narrative depth that also has a desktop companion mode as a bonus feature.
