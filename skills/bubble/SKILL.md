# Bubble

When Josh says "bubble \<node\>", add a `[[link]]` to that node in a higher
branch's index file. The linked node stays at its current depth; only the
surface ref moves up. One level at a time.

**Example.** `feedback_free_worktree_branch_then_switch_in_place` lives under
`feedback_sending`. A `[[feedback_free_worktree_branch_then_switch_in_place]]`
line in `feedback_dispatch_process.md` surfaces it one level up.
