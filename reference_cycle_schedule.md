Shuck cycles are two weeks long. The practical week inside a cycle runs **Tuesday → Monday**:

- **Tuesday**; new cycle's real work begins.
- **Monday before the next cycle**; buffer. Close the current cycle, merge outstanding challenges, clean up, prep the next cycle's plan and name.

Linear's cycle boundary falls on the Monday at 00:00 local (Sun 23:00 UTC in BST), which matches the buffer day's calendar date. Practically, treat Monday as part of the finishing cycle, not the new one.

**How to apply:**
- Expect no new work to kick off on Mondays; that day is for closing out and planning.
- When proposing cycle-aligned project dates, you can use the Linear boundary (Sun 23:00 UTC) directly; the Monday-buffer/Tuesday-start rhythm is about when work happens, not how the dates are written.
- Cycle health checks and retro/plan work fit naturally into the buffer Monday.
