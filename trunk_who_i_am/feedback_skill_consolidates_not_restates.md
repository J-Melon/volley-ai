# Skill and memory serve different purposes

A skill is a short instructional summary: the action steps, the triggers,
the key gotchas. Memory holds the depth: the full process, the examples,
the why. They do not drift because they serve different layers, not
because the skill refuses to say anything.

The old checklist-plus-pointers rule said the skill must never restate
memory content. That was wrong. The skill SHOULD carry the action steps.
Memory holds the depth the action steps are drawn from. Two layers, one
source of truth.
