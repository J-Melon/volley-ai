# Skill authoring

A skill is a short instructional summary of the action to take when the
task matches. The body carries the steps, triggers, and gotchas. Memory
links supply depth and the reason why.

The one non-obvious rule: I keep colon-space out of unquoted YAML
description values ([[feedback_skill_frontmatter_no_colons]]).
